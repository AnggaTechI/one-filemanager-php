# one-filemanager-php

A lightweight, single-file PHP file manager designed for quick deployment and simple web-based file management.  
Built to be portable and easy to use, `one-filemanager-php` can run on environments ranging from **PHP 5.4 up to the latest PHP versions**.
Pass : admin1230

## Overview

`one-filemanager-php` is intended for users who need a straightforward way to manage files and directories directly from a browser without installing a full application stack or additional dependencies.

It is especially useful for:

- Rapid deployment on shared hosting or VPS environments
- Emergency file access and maintenance
- Lightweight file operations without FTP clients
- Simple server-side file browsing and editing

## Features

- Single-file deployment
- No external dependencies
- File and directory browsing
- File upload and download
- File and folder rename
- File and folder deletion
- Create new directories
- Edit text-based files directly in the browser
- Simple and lightweight interface
- Broad compatibility across PHP versions
- etc

## Requirements

- PHP **5.4 or newer**
- A web server such as Apache, Nginx, or OpenLiteSpeed
- Read/write permissions for the target directories

## Compatibility

This project supports:

- **PHP 5.4**
- PHP 5.5+
- PHP 7.x
- PHP 8.x
- **Latest stable PHP releases**
